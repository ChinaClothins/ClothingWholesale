# 部署到阿里云服务器

## 你的网站是纯静态的，部署只需 3 步：装 Nginx → 传文件 → 配站点

---

## 一、登录服务器

打开 PowerShell，执行：

```
ssh root@你的服务器IP
```

---

## 二、安装 Nginx

```bash
# CentOS
yum install -y nginx

# Ubuntu
apt update && apt install -y nginx

# 启动
systemctl start nginx
systemctl enable nginx
```

---

## 三、开放 80 端口

### 阿里云安全组（必须做）
1. 阿里云控制台 → ECS → 安全组 → 配置规则
2. 添加入方向规则：TCP / 80 / 0.0.0.0/0

### 服务器防火墙
```bash
# CentOS
firewall-cmd --permanent --add-port=80/tcp
firewall-cmd --reload
```

---

## 四、创建站点目录

```bash
mkdir -p /var/www/everstock
```

---

## 五、上传文件

在你本地电脑 PowerShell 中，进入项目目录：

```powershell
cd "E:\LuoZifeng\项目\test\企业官网(便捷上货版本)\服装批发"
scp -r img products.csv products.js 电商.html root@你的IP:/var/www/everstock/
```

---

## 六、配置 Nginx

在服务器上：

```bash
vi /etc/nginx/conf.d/everstock.conf
```

粘贴：

```nginx
server {
    listen       80;
    server_name  _;
    root         /var/www/everstock;
    index        电商.html;

    location / {
        try_files $uri /电商.html;
    }
}
```

保存退出后：

```bash
nginx -t
systemctl restart nginx
```

---

## 七、验证

浏览器打开 `http://你的IP` 即可。

---

## 日常更新

改完 CSV → 双击 `更新预览.bat` 确认 → 上传：

```powershell
cd "E:\LuoZifeng\项目\test\企业官网(便捷上货版本)\服装批发"
scp products.csv products.js root@你的IP:/var/www/everstock/
```

新增图片也要上传：

```powershell
scp -r img root@你的IP:/var/www/everstock/
```
